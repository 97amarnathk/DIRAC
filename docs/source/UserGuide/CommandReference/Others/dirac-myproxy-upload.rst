====================
dirac-myproxy-upload
====================

Usage::

  dirac-myproxy-upload.py (<options>|<cfgFile>)*

Options::

  -f  --file <value>           : File to use as proxy
  -D  --DN                     : Use DN as myproxy username
  -i  --version                : Print version
